# Coffee-sales-analysis
This dashboard analyzes coffee shop sales data using Power BI. It covers total customers (3.64K), revenue (115.43K), product performance across 8 coffee types, and payment method breakdown. Americano with Milk is the top seller, and 97.55% of customers pay with cash.


#  Coffee Sales Analysis Dashboard

A Power BI dashboard providing visual insights into coffee shop sales performance, customer trends, and payment behavior.
